# notes-nodejs

Sample NodeJS application to demonstrate REST API design with ExpressJS.

#### To run -

    $ npm start

#### To run during development (with active watch and changes reflect real time) -

    $ npm run start-dev

#### To clean the project -

    $ npm run clean

#### To build the project -

    $ npm run build
